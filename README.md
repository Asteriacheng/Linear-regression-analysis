# Linear-regression-analysis
In this project, linear regression analysis is applied to investigate the relationship between KS4 examination performance and a set of explanatory variables. The model is constructed using data from a linked dataset combining the Next Steps Study (LSYPE) and the National Pupil Database (NPD). Prior KS3 attainment, demographic characteristics, family background, behavioural indicators, and educational attitudes are included as predictors. The sample consists of 13,448 pupils. 

The linear regression model is estimated to identify statistically significant factors associated with KS4 scores, with variable selection based on significance testing. Standard diagnostic checks are conducted to assess model validity, and the results show that prior KS3 attainment is the most influential predictor of KS4 performance among all variables considered.
