# Linear-regression-analysis
This project applies linear regression to analyse factors associated with KS4 examination performance. Exploratory data analysis was first conducted to examine relationships between KS4 scores and key categorical variables. Several categorical predictors were merged into broader meaningful groups, and missing string values were recoded. A backward elimination approach was then used to select significant predictors from a full regression model. Model diagnostics, including residual analysis, multicollinearity assessment using VIF, and influence statistics, were performed to evaluate model validity.
