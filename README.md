# Linear-regression-analysis
This project examines the factors associated with pupils’ Key Stage 4 (KS4) examination performance at age 16. The analysis focuses on understanding how prior academic achievement, demographic background, family characteristics, and behavioural factors relate to differences in KS4 outcomes. The study is based on a linked dataset combining the Next Steps Study (formerly LSYPE) and the National Pupil Database (NPD). The Next Steps Study tracks approximately 16,000 young people in England born in 1989–90 from age 14 onwards, while the NPD provides matched examination records at Key Stage 3 (age 14) and Key Stage 4 (age 16). The merged dataset contains detailed information on prior attainment, socioeconomic status, school behaviour, and educational attitudes. The working sample used in this analysis includes 13,448 pupils. Regression analysis shows that several factors are significantly associated with KS4 performance, including prior KS3 attainment, gender, housing tenure, area deprivation (IDACI), pupil and parental aspirations, attitude towards school, homework frequency, truancy, school exclusion, special educational needs (SEN), free school meal eligibility, and mother’s highest qualification, with prior KS3 attainment emerging as the strongest predictor of KS4 achievement.
